About the Project

Background information, architecture, and technology specs of the credit card fraud detection system.

Project Summary

Credit card fraud causes billions of dollars in losses annually. This project demonstrates how neural architectures and ensemble models can run real-time inference on anonymized transaction data. By applying SMOTE (Synthetic Minority Over-sampling Technique) during the training phase, we resolved severe class imbalances (fraud representing only 0.49% of raw records).

The core modeling leverages XGBoost (with an F1 of 14.29% and ROC-AUC of 0.7981) and a multi-layer deep network in TensorFlow/Keras. Features V1-V28 represent principal PCA components of cardholder sensitive information to preserve privacy, and Amount/Time metadata values are scaled according to training distributions.
